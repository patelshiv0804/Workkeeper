# 🚀 WorkKeeper – Employee Management & Attendance System

WorkKeeper is a full-stack web application designed to manage employee attendance, leave requests, and organizational workflows efficiently. It provides a centralized platform for tracking employee activities, managing roles, and generating insights through dashboards.

## 🛠️ Tech Stack

### Frontend
- React.js  
- HTML, CSS, JavaScript  

## ⚙️ Prerequisites

Make sure you have installed:

- Node.js (v16 or higher)  
- npm   
- Git
